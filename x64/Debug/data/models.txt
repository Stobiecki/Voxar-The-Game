Here is introduction to loaded models!

It should end with 'at' symbol.

What type of models can be loaded:

SM - Standard Model					DONE
TM - Transparent Model					TO_DO
RM - Reflective Model					TO_DO
TRM - Transparent and Reflective Model			TO_DO
2D - 2D Model						DONE
2DC - 2D with Crossed textures				DONE
3DA - 3D with simple Animations				TO_DO

order:

<model_type>;<model_name>;<model_path>;<texture>;<program_name>;<vertex_shader>;<fragment_shader>;
